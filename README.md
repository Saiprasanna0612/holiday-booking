# 🌍 Gogaga Holidays – Travel Booking Web App

## 📌 Project Overview

This project is a **responsive Holiday Package Booking System** built using **HTML, CSS, and JavaScript**.

It allows users to:

* Search for destinations (National & International)
* View holiday packages
* Explore hotels with ratings
* Check day-wise travel itinerary
* Compare travel options (Flight, Bus, Car)
* Book packages and travel options

---

## 🚀 Features

### 🔍 Travel Planner

* Select **Trip Type (National / International)**
* Dynamic **Destination Dropdown**
* Budget & Traveler input
* Displays:

  * Package details
  * Hotels with ratings
  * Day-wise itinerary
  * Travel options with price & time

### 🏨 Package Details
